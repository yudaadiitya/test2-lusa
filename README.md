# WARUNG MAKAN APP

## Set Path to where File is Located (using cd).
```
cd warung-makan-client
```
```
cd warung-makan-api
```
## Project setup
```
npm install
```

### Compiles and hot-reloads for development Client
```
npm run serve
```

### Running Server
```
npm start
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
